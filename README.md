# JavaTraining-Week3
タスク管理アプリを作成しよう

「アプリの概要」
本アプリは、コマンドライン上でタスクの登録・一覧表示・削除・完了ができるToDo管理アプリです。
番号を入力して操作を選択することで、簡単にタスク管理を行うことができます。

「実行方法」
1,ターミナル（またはコマンドプロンプト）で、プログラムのあるフォルダに移動します。
2,次のコマンドでコンパイルします。
`javac Main.java`
3,コンパイルが成功したら、以下のコマンドで実行をします。
`java Main`

「工夫した点・苦労した点」
・削除するときや完了にするときに、タイトル名を一致させなければならなかったので、入力の際にはタスクの一覧を表示させるようにしました。
・タスクが存在していない状態で「タスク一覧を表示」「タスクを削除」「タスクを完了にする」を選択した場合に、メニュー一覧に戻るようにしました。
・try-catchを使うべきか、ifで処理するべきかを迷う場面がありました。可能な限りtry-catchを使用したのですが、finallyに関しては使用する場面が難しいと感じ、結局使うことができませんでした。
